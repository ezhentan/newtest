About Me

My name is E-Zhen Tan, a fresh graduate currently working as a Data Analyst at NCS Group. I love discovering the versatility of code, from using Python to create dictionaries (actual ones, not the data type '{}') to creating websites with R.

Technical Skills

Having studied Business Analytics in university and from my previous internship, I have gained experience in using Python to analyse data and build machine learning models. My previous stint as a Data Science Intern gave me real-world experience in using Python for machine learning, and since then, I have continued pursuing this area of interest by taking the Deep Learning Specialization on Coursera. The types of projects I have done include:

  • Text classification
  • Image classification
  • Sentiment analysis
  • Web scraping with BeautifulSoup

In addition to Python, I am able to create visuals and graphs of present and historical data using Tableau. Equipped with these skills, I am able to manage data and assist in giving meaning to the vast sea of data available. My portfolio of prior projects are linked here and in the tab, GitHub, above.

I am fortunate and grateful to have been able to gain experience from 3 different internships in 3 different domains - Data Science Intern at Jewel Paymentech, Data Analyst Intern at Shopback, and International Business Development Intern at Love, Bonito. Each internship taught me invaluable lessons, from technical skills to better workplace communication to business mindsets.

A self-directed learner at heart, I have also pushed myself to learn other codes beyond what is taught in the school curriculum. In building this website using R, I have also used the HTML language to format the text.
